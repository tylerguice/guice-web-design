# Guice Web Design

Static one-page website for Guice Web Design.

## Run Locally

```bash
npm run dev
```

Then open:

```text
http://localhost:5173
```

## Deploy

This project is intentionally static. It can be deployed to Cloudflare Pages by connecting the GitHub repo and using:

- Build command: leave blank
- Build output directory: `/`

## Notes

The contact form currently shows a local success state only. Connect it to Cloudflare, Formspree, or another email service before using it for real leads.
